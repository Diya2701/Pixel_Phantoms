DOMINATION_METRIC (MCS)

The Multidimensional Contribution Score (MCS) ensures that complex engineering is rewarded over simple commits.

  • Level 3 (Architect): 1100 XP - Major core features, security patches, or infrastructure.
  • Level 2 (Specialist): 500 XP - Significant bug fixes, UI improvements, or module development.
  • Level 1 (Scout): 200 XP - Documentation, minor CSS fixes, or README updates.

SECURITY_ENFORCEMENT

To maintain codebase integrity, all candidates must adhere to automated protection protocols.

  • Linting: Every Pull Request must pass ESLint/Prettier checks automatically.
  • Security: Secrets or API keys detected in commits lead to immediate disqualification.
  • Review: 2-peer approval is mandatory before any "Elite" class PR is merged.

CANDIDATE_ELIGIBILITY

Participation is open to all students, but advancement requires verified skills.

  • Titan Class: Requires 15,000+ XP and 5 successfully merged L3 PRs.
  • Striker Class: Requires 5,000+ XP and active community mentoring.
  • Rookie: Initial entry level for all new registered GitHub handles.

EVENT_TRANSPARENCY

We value real-world presence as much as code. Our attendance tracking is verified via cryptographic proof.

  • Standard Reward: 250 XP for every verified campus session.
  • Bonus Impact: Hosting an event grants an automatic 1.5x multiplier to your next PR.
